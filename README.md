# 图书管理系统

复旦大学数据库课程中期实验——基于 Flask 的图书管理系统

## 项目简介
一个图书管理系统，支持图书管理、进货管理、销售管理、财务统计、用户管理等功能。

前端使用纯HTML+CSS+JavaScript（含Chart.js可视化图表），后端使用Flask+SQLite数据库。

## 项目结构

```
├── app.py          # Flask主应用（路由+页面渲染）
├── database.py     # 数据库连接与初始化
├── user.py         # 用户管理逻辑
├── book.py         # 图书管理逻辑
├── purchase.py     # 进货管理逻辑
├── sale.py         # 销售管理逻辑
├── bill.py         # 账单管理逻辑
├── log.py          # 日志管理逻辑
├── sample_books.csv   # 示例图书数据
├── sample_users.csv   # 示例用户数据
├── bookstore.db       # SQLite 数据库文件（运行后自动生成）
└── README.md
```

## 数据库表结构

| 表名 | 说明 |
|------|------|
| `user` | 用户信息（含角色权限） |
| `book` | 图书信息（ISBN、书名、作者、出版社、零售价） |
| `inventory` | 库存信息 |
| `purchase` | 进货单（含状态：未付款→已付款→已入库 或 未付款→已退货|
| `sale` | 销售记录 |
| `bill` | 财务账单（收入/支出）和图表展示|
| `log` | 操作日志 |


## 菜单栏介绍

### 用户管理
- 超级管理员可添加/修改/删除普通管理员
- 支持CSV批量导入用户
- 用户角色：超级管理员、普通管理员
- 个人中心：修改个人信息、修改密码

### 图书管理
- 超级管理员可添加、修改、删除图书信息，支持CSV批量导入/导出
- 支持 ISBN、书名、作者、出版社、价格区间、库存数量等多种查询方式，支持模糊搜索
- 展示所有图书列表


### 进货管理
- 支持已有图书进货和添加新书进货两种方式，仅超级管理员账号支持添加新书进货
- 进货记录查询
- 进行付款、退款、入库操作；入库时需给新书设置零售价

### 销售管理
- 对有库存的书籍进行销售
- 销售记录查询，支持按日期筛选
### 库存管理
- 书籍基本信息和库存数量展示
- 随进货、销售操作变化
- 图书库存不足预警
- 支持按照书籍id、ISBN、书名、作者名或出版社查询
- 超级管理员可对图书信息进行修改
### 财务统计
- 自动记录所有收支账单
- 月度收支趋势图（柱状图+折线图）
- 收支占比饼图
- 图书销售排行榜（金额/数量柱状图各一个）
- 支持按时间段筛选

### 操作日志
- 记录所有用户的操作行为
- 支持按用户名、操作类型、时间范围筛选
- 普通管理员仅查看自己的日志，超级管理员可查看全部日志

## 运行项目

### 1. 安装依赖

```bash
pip install flask
```

### 2. 启动系统

```bash
python app.py
```
自动创建初始化数据库，数据保存在创建的bookstore.db文件中。

默认访问地址：`http://localhost:5000`

### 3. 默认管理员账号

| 用户名 | 密码 | 角色 |
|--------|------|------|
| `admin` | `123456` | 超级管理员 |

登录后可创建更多普通管理员账号。
